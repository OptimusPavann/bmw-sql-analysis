# BMW Sales SQL Analysis

This project analyzes a BMW vehicle sales dataset using SQL.

## Dataset Columns
- model
- year
- region
- color
- fuel_type
- transmission
- engine_size_l
- mileage_km
- price_usd
- sales_volume

## SQL Topics Covered
- Basic SQL queries
- Aggregations
- GROUP BY and HAVING
- Subqueries
- Window Functions
- Business Analysis Queries
